This practical work is based on two articles:

Errors and Complications in SQL Query Formulation
Toni Taipalus, Mikko Siponen & Tero Vartiainen

What to Expect and What to Focus on in SQL Query Teaching
Toni Taipalus & Piia Perälä

Databases and tables where created with pgAdmin 4 and it uses PostgreSQL.
The example data used in the database was created in Mockaroo.com website (Random Data Generator).
In this work, I had to design and create a database. The articles had exercises which are shown in the exercises.PNG. 
This practical work was part of an another study and the courses databases and exercises' answers where part of the study.
